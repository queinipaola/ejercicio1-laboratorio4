Elementos de un sistema de informaciòn computacional

Equipo computacional: Hardware necesario para que el sistema de informaciòn funcione, integrado por computadoras y equipo periférico conectadol a ella.

Recurso humano: Son las personas que intertúan con el sistema de informaciòn, suministrar datos o utilizan el sistema para generar datos.

Fuentes:Son los datos e informacion que se introducen en el sistema,es decir,son las entradas requeridas para que el sistema informático funcione.

Programas:Software ejecutado por la computadora, el cual procesa la informaciòn de entrada para generar los recultados esperados.

Telecomunicaciones:Abarca el software y hardware que permite la transmiciòn en forma electrònica de texto e imágenes, entre otros.

Politicas y reglas de operaciòn:Aplicadas a los procedimientos y mecanismos para hacer que una aplicaciòn de computadoras trabaje de acuerdo con estas o regir la partde funcional del proceso admistrativo de la empresa.
